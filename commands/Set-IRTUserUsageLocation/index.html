    Set-IRTConfig

    SYNOPSIS

    Interactively updates IRT configuration settings.

    SYNTAX

    Set-IRTConfig [-Reset] [-ProgressAction <ActionPreference>] [-WhatIf] [-Confirm] [<CommonParameters>]

    DESCRIPTION

    Presents a menu of configuration settings. When the user selects a setting, shows a description and available options, then saves the new value.
